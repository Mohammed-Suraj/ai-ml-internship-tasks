# Day 6: K-Nearest Neighbors (KNN) Classification

## Objective:
Understand and implement KNN for classification problems.

## Steps Performed:
- Loaded Iris dataset
- Selected features and target labels
- Split dataset into training and testing sets
- Applied feature normalization using StandardScaler
- Trained KNN model with different K values
- Selected best K based on accuracy
- Evaluated model using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report
- Visualized:
  - Accuracy vs K graph
  - Decision Boundary

## Tools Used:
- Python
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
